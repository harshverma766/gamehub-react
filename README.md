# 🎮 GameHub

Modern Game Discovery Web App built with React & Vite featuring search, filters, wishlist, and dark mode.

---

## 🚀 Live Demo

https://gamehub-react-henna.vercel.app/

---

## ✨ Features

- 🔍 Search games
- 🎯 Filter by genre
- ❤️ Wishlist functionality
- 🌙 Dark / Light mode
- 📊 Game ratings
- ⚡ Fast performance (Vite)

---

## 🛠 Tech Stack

- React
- Vite
- JavaScript
- Context API
- CSS
